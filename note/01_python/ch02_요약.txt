파이썬 자료형과 연산자 요약(1~3절)
1절 변수

파이썬은 선언 없이 할당만으로 변수 생성 및 타입 자동 지정됨
변수명 규칙: 문자, 숫자, 언더스코어(_) 사용 가능, 숫자로 시작 불가, 예약어 사용 불가
다중 변수 할당 지원: a, b = 10, 20 (일괄 동시 할당)
변수 값 교환 간편: a, b = b, a (swap)
help() 함수로 도움말 확인 가능

2절 화면 입출력(I/O)

입력: input() 함수는 항상 문자열로 값 반환

숫자로 변환 필요 시: int(input()), float(input())


출력: print() 함수 활용

구분자 지정: sep='구분자' (기본값: 공백)
끝문자 지정: end='문자' (기본값: 줄바꿈 \n)
출력 방향 지정: file=파일객체 (기본값: sys.stdout)



3절 기본 자료형

숫자형

정수형(int): 크기 제한 없음, 언더스코어로 구분 가능 (1_000_000)
실수형(float): 3.14159, 2.71828
복소수형(complex): 2+3j
특징: 정수/정수 나눗셈은 항상 실수 반환 (5/2 → 2.5)
정수 결과 얻기: 몫 연산자 (//) 또는 int() 함수 사용
지수 표현법: 1.12e12 (큰 수), 1.12e-12 (작은 수) → 실수형으로 저장


문자형(str)

작은따옴표(') 또는 큰따옴표(") 사용
연산: +(연결), *(반복)
인덱싱: name[0], name[-1] (음수는 뒤에서부터)
슬라이싱: name[0:2] (부분 문자열 추출)
여러 줄 문자열: 따옴표 세 개(''' 또는 """) 사용
원시 문자열(Raw string): r'string' (이스케이프 문자 무시)


논리형(bool)

True 또는 False 두 값만 가짐
비교 연산자: >, <, ==, != 등
논리 연산자: and, or, not
